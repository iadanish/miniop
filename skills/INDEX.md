## Usage in MiniOp

Each skill is used in specific parts of the MiniOp pipeline:

1. **Video Upload/URL**: miniop-video-processing
2. **AI Analysis**: miniop-ai-clipping
3. **Caption Generation**: miniop-captions
4. **Brand Application**: miniop-brand
5. **Social Posting**: miniop-social
6. **Frontend UI**: React, Next.js, Design skills
7. **Database**: Supabase, Neon skills
8. **Testing**: Playwright, TDD skills
9. **Agent Orchestration**: Agent workflow skills
10. **Video Rendering**: Remotion skills
